# jetson_team_c

## 使い方

USBカメラを繋げてJetsonを起動。
HDMIでJetsonをディスプレイに繋ぐ。

Jetson上で以下のコマンドを実行する。

このリポジトリをクローン
```
git clone https://github.com/HaseU-git/jetson_team_c/
cd jetson_team_c
```

Dockerを起動するところまで
（時間がかかる）
```
chmod 744 run_project.sh
./runproject.sh
```

Docker上でプログラムを実行するまで
（時間がかかる）
```
chmod 744 start_up.sh
./start_up.sh
```
